# Bluegrass Valentine – Promotional Website

## Overview
A professional, lightweight landing page for the San Jose-based bluegrass band, "Bluegrass Valentine". Designed for fast loading and easy access to live recordings and contact information.

## Features
- Music Playback: Optimized, click-to-load audio player for live recordings.
- Promotional Layout: Clean, mobile-responsive design highlighting band photography and music.
- Media Optimization: Audio tracks and images optimized for fast performance.

## Technologies
- HTML/CSS/JS: Pure, framework-free implementation for maximum performance.
- Netlify: Deployed for reliable, global, and fast content delivery.
- FFmpeg: Used to optimize audio file size while maintaining quality.
- AI-Assisted Workflow: Developed using an AI-assisted workflow for rapid prototyping and iterative improvements.
